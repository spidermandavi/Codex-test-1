# Racing Kings Chessground Demo

A static Racing Kings demo that embeds the Lichess Chessground board from a CDN and adds lightweight variant move logic for legal-square hints, highlights, premoves, and an automatic Black reply.

Chessground is GPL-3.0-or-later software; keep that license in mind if you distribute this page or combine it with other code.

Open `index.html` in a modern browser or serve the folder with any static web server.
